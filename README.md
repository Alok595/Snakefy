# 🐍 Snakefy

AI-powered snake detection and emergency assistance system.

## 📖 About

**Snakefy** is an intelligent web application designed to identify snakes from images and provide real-time safety guidance. Using advanced AI, it helps users determine whether a snake is venomous and offers first-aid instructions along with nearby hospital suggestions.

Built especially for real-world safety, Snakefy aims to assist users in critical situations with fast and reliable information.

---

## ⚡ Features

* 🧠 **AI Snake Detection** (Image-based)
* ⚠️ **Venomous / Non-venomous Classification**
* 📊 **Confidence Score & Danger Level**
* 🏥 **Nearby Hospital Finder (Location-based)**
* 🩹 **First Aid Instructions**
* 🌍 **Multi-language Support**
* 📷 **Camera + Image Upload**

---

## 🚀 How It Works

1. Capture or upload a snake image
2. AI analyzes the image
3. Get instant results:

   * Snake species
   * Venom status
   * Danger level
4. View first-aid steps
5. Find nearby hospitals (if needed)

---

## ⚠️ Important Disclaimer

Snakefy provides **AI-based predictions**, which may not be 100% accurate.
Always:

* Keep a safe distance
* Avoid handling snakes
* Seek professional help immediately
